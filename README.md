# BAW
Celem tego projektu było zapoznanie się z podatnościami z OWASP Top 10.

Aby wykonać ten kod potrzebne są:
Python3
Biblioteki do python:
  Selenium
  Time
Robot Framework

Dodatkowo, należy uruchomić aplikację WebGoat na domyślnych portach wdedług instrukcji ze strony projektu: https://github.com/WebGoat/WebGoat

Aby uruchomić kod należy wykonać polecenie: robot <ścieżka_do_pliku_TestCases.robot>
"Suite setup" należy wykonać tylko przy pierwszym uruchomieniu kodu.

The purpose of this project was to learn about vulnerabilities from the OWASP Top 10.

To execute this code you will need:
Python3
Libraries for python:
  Selenium
  Time
Robot Framework

In addition, you need to run the WebGoat application on the default wd ports according to the instructions from the project page: https://github.com/WebGoat/WebGoat

To run the code, execute the command: robot <path_to_file_TestCases.robot>.
"Suite setup" should be performed only when running the code for the first time.
